## Performance Tips

1. **Lazy Loading:** Use `loading="lazy"` attribute (already included above)
2. **Async:** The widget loads independently - won't block your page
3. **CDN:** Widget uses Google Fonts CDN for fast loading
4. **Size:** Widget is <50KB total (HTML + CSS + JS)

- Extended: `height="750px"`

- Narrow: `max-width: 400px`

## Troubleshooting

### Widget Not Showing
- Check that the URL is correct: `https://quantummerlin.com/tools/angel-number-micro.html`
- Verify iframe isn't blocked by CSP (Content Security Policy)
- Check browser console for errors

### Widget Too Tall/Short
- Adjust `height` attribute to fit your layout
- On mobile, widget adapts but may need 700-750px height

### Styling Conflicts
- Widget is self-contained and shouldn't conflict
- Use `border: none` and `border-radius` for clean integration
