This directory stores all article images — OG cards, hero images, Pinterest pins, and in-article screenshots.

**Full visual guide:** See `agents/article-visual-guide.md` for detailed image descriptions for all 70 articles.

## File Naming Convention

```
{slug}-hero.jpg       → Hero image (1200×675) — main article visual
{slug}-og.jpg         → OG/Social card (1200×630) — social share preview
{slug}-pin.jpg        → Pinterest pin (1000×1500) — vertical social
{slug}-screenshot-N.jpg → In-article screenshots
```

## Brand Palette

- Primary gradient: #667eea → #764ba2
- Background: white or very light gray (#fafafa)
- Text: dark (#1a1a2e or #2d2d2d)
