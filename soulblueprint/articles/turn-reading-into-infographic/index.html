Build It in Canva (Free)

Open Canva (free account works perfectly) and follow these steps:

  1. Search "infographic" in Canva's template library
  2. Pick a template that matches your chosen layout (vertical flow is easiest)
  3. Replace the placeholder text with your AI-extracted insights
  4. Update colors to match the AI's color suggestion (or use the purple/blue Soul Blueprint palette: #667eea to #764ba2)
  5. Add the person's name as the headline
  6. Add the strengths as icon-labeled bullet points
  7. Add the quote in a highlighted box at the bottom

Common Mistakes to Avoid

  1. Cramming too much text. An infographic should be 80% visual, 20% text. If it looks like a document, simplify.
  2. No visual hierarchy. The headline should be biggest, strengths should be scannable, and the quote should pop. Don't make everything the same size.
  3. Forgetting mobile. Most people view on their phone. Use large fonts (minimum 14pt for body text) and high-contrast colors.
  4. Using the whole reading. Pick the 5-8 most striking insights. The AI prompt in Step 2 does this for you automatically.
  5. Ignoring Pinterest dimensions. For Pinterest, always go vertical (1000√ó1500 or 1000√ó2100). Horizontal infographics get almost no visibility.
