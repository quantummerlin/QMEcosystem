How This Page Works

This page is built using Eleventy (11ty) with Nunjucks templates. Instead of duplicating the same boilerplate HTML across every page, it uses a shared layout that provides:

  • ✨ Google Analytics (GA4) — auto-injected
  • ✨ AdSense — auto-injected
  • ✨ Open Graph & Twitter Cards — from front matter
  • ✨ Canonical URL — auto-generated
  • ✨ Etsy CTA — auto-injected
  • ✨ Footer with legal links — from layout

For Developers

To create a new page using this template system, create a .njk file in src/ with front matter: 

---
layout: qme.njk        # or stranger.njk
title: "My New Tool"
description: "Tool description for SEO"
permalink: /my-tool/
---

<!-- Your tool HTML goes here -->
<div class="tool-card">...</div>

Build with npx eleventy — output goes to _site/.
